# ZILKIT

A monorepo containing packages for building Next.js applications with Sanity CMS.

## Packages

### @zilfire/core-theme

A comprehensive React component library and type system for building Next.js applications with Sanity CMS integration.

**Features:**

- Pre-built page blocks (Hero, FAQ, Features, Media Content, Header, Footer)
- Reusable UI components (Button, Section, Container, etc.)
- Typography components with consistent styling
- Sanity schema definitions that map to React components
- Full TypeScript support with type-safe styling system
- Tree-shakeable modular exports

> ⚠️ **Warning: Active Development**
>
> Core Theme is under active development and the API is subject to change. Breaking changes may occur between versions until the beta release. **Beta launch is expected in Q2 2026.**

[View Core Theme Documentation →](./packages/core-theme/README.md)
